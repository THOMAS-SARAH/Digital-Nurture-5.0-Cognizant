# Spring Learn

## Project Description

This project demonstrates the creation of a basic Spring Boot Web Application using Maven.

The application is created using Spring Initializr with the following configuration:

- Group: com.cognizant
- Artifact: spring-learn
- Java Version: 17
- Build Tool: Maven
- Spring Boot: 3.x

---

## Technologies Used

- Java 17
- Spring Boot
- Maven
- Spring Web
- Spring Boot DevTools
