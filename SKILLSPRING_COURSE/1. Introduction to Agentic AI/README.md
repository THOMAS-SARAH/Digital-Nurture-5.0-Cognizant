# Introduction to Agentic AI - Course Summary

## 1. Traditional AI vs Modern AI

### Definition
- Traditional AI follows predefined rules and performs specific tasks.
- Modern AI uses Machine Learning (ML), Deep Learning (DL), Large Language Models (LLMs), and AI agents to learn, reason, and make decisions.

### Key Points
- Traditional AI is rule-based and task-specific.
- Generative AI creates new content such as text, images, code, and audio.
- Agentic AI extends Generative AI by autonomously planning, reasoning, using tools, and executing tasks.
- Agentic AI can interact with external systems and adapt based on feedback.

### Important Terms
- **Traditional AI:** Rule-based intelligent systems.
- **Generative AI:** AI that generates new content.
- **Agentic AI:** AI capable of autonomous decision-making and task execution.
- **LLM (Large Language Model):** Foundation model powering Generative AI.

### Interview Points
- Every Agentic AI system may use Generative AI, but not every Generative AI system is Agentic.
- Agentic AI focuses on **goal completion**, not just **content generation**.

# 2. Core Concepts and Architecture of Agentic AI

## Definition
- Agentic AI combines intelligence, planning, memory, reasoning, and tool usage to solve complex problems autonomously.

## Key Points
- Receives a goal from the user.
- Plans multiple steps to achieve the goal.
- Uses memory to retain context.
- Calls external tools and APIs when needed.
- Evaluates results and improves responses iteratively.

## Important Components
- Goal
- Planning
- Memory
- Reasoning
- Tool Use
- Execution
- Feedback Loop

### Interview Points
- Agentic AI architecture follows a **sense → think → act → learn** cycle.

---

# 3. Role of Generative AI in Agentic AI

## Definition
- Generative AI acts as the intelligence engine that enables Agentic AI to understand, generate, and communicate information.

## Key Points
- Generates responses and content.
- Understands user instructions.
- Assists in reasoning and planning.
- Produces code, documents, and summaries.
- Supports interaction with users.

## Advantages
- Natural language understanding.
- High-quality content generation.
- Supports multiple domains.

## Interview Points
- Generative AI provides intelligence, while Agentic AI adds autonomy and execution.

---

# 4. AI Agents and Their Lifecycle

## Definition
- An AI Agent is an autonomous software entity that perceives its environment, reasons, plans, acts, and learns to achieve a goal.

## AI Agent Lifecycle

1. Goal Definition
2. Perception (collect inputs)
3. Reasoning
4. Planning
5. Action Execution
6. Observe Results
7. Feedback
8. Continuous Improvement

### Key Points
- Agents continuously interact with their environment.
- Can use APIs, databases, web search, and software tools.
- Maintain memory across tasks.
- Improve decisions using feedback.

### Interview Points
- AI agents operate autonomously with minimal human intervention.

---

# 5. Agentic AI Workflow

## Definition
- Agentic AI follows an iterative workflow to complete complex tasks intelligently.

## Workflow Steps

### Perception
- Understands user input.
- Collects information from the environment.

### Reasoning
- Analyzes the problem.
- Chooses the best strategy.

### Planning
- Breaks the task into smaller steps.
- Determines execution order.

### Action
- Uses tools, APIs, or software.
- Executes planned tasks.

### Feedback
- Evaluates outcomes.
- Refines future decisions.
- Repeats the cycle if necessary.

### Interview Points
- Feedback makes Agentic AI adaptive and self-improving.

# 6. Advanced Agentic AI Concepts

## Function Calling

### Definition
- Enables LLMs to invoke external functions or APIs to perform real-world tasks.

### Examples

- Calendar API
- Payment API
- Database Query

---

## Stateful AI Agents

### Definition
- Agents that remember previous interactions and maintain context over time.

### Advantages
- Personalized responses.
- Better long-term conversations.
- Efficient task continuation.

---

## Multi-Agent Systems

### Definition
- Multiple AI agents collaborate to solve a larger problem.

### Advantages
- Parallel task execution.
- Specialized responsibilities.
- Better scalability.

---
